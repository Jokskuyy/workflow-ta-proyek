# Panduan Download Manual Referensi Jurnal

File ini berisi link untuk mendownload manual jurnal referensi yang tidak bisa didownload otomatis
karena jurnal OJS Indonesia umumnya memblokir akses bot/script.

## Cara Download

1. Klik link artikel di kolom "Link Artikel" 
2. Di halaman artikel, klik tombol **"PDF"** atau **"Download"**
3. Simpan file PDF ke folder: `journal/references/`
4. Ganti nama file sesuai kolom "Nama File"
